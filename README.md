### How to set up?

- Setup environment files (provided by repo owners)

  - .env

- Install the dependencies and devDependencies.

```sh
$ cd your-project
$ yarn install
$ cd packages/api
$ yarn install
```

---

### Development

```sh
$ cd your-project/packages/api
$ yarn dev
```

### JavaScript style guide, linter, and formatter

### Test Process

- run tests

```sh
$ cd your-project/packages/api
$ yarn test
```

- run tests coverage

```sh
$ cd your-project/packages/api
$ yarn test:coverage
```

### Build Process
